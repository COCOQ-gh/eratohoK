∇eratohoK Ver0.161用 モブ見分けパッチ

▲概要
通し番号がオミットされたので今度はアルファベットを付けてみた
野盗だけ贔屓にしたら外来人とホフゴブリンが襲ってきそうだったのでそちらにも対応した


■パッチ内容
・NEWGAME.ERB
　@BANDIT_INIT
・GOBLIN_FUNCTIONS.ERB
　@GOBLIN_INIT
・GAIRAI_FUNCTION.ERB
　@GAIRAI_INIT
　モブ作成時に名前の後ろにアルファベットが付くように設定

■追記内容「%UNICODE(0xFF21 + LCOUNT)%」の説明
　%UNICODE(0xFF21 + LCOUNT)%：%〜%内を文字列式として処理
　UNICODE(0xFF21 + LCOUNT)：UNICODEの該当コードの文字を返す関数
　0xFF21 + LCOUNT：0xFF21(Ａを示すコード番号)とLCOUNT[キャラ作成時のループカウント]の合計値
LCOUNTは0から1ずつ増加するため、Ａから順にアルファベットがついていきます。

・参考
半角アルファベット大文字：0x0041(A)
半角アルファベット小文字：0x0061(a)
全角アルファベット大文字：0xFF21(Ａ)
全角アルファベット小文字：0xFF41(ａ)

▲パッチ適用方法
eratohoK 0.161に、このパッチを上書きしてください
