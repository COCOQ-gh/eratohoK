
eratohoK 0.232.1用 スカウト時に金額を入力する手間を省きたいパッチ

 0[支払う]
 1[やめる]
ではなく
 0[要求額を支払う]
 1[支払い金額を入力する]
 2[やめる]
の3択にするのよ。
0にすると、金額を入力する手間が省けるのよ。
1にすると金額を入力できるけど、いくら支払う……？ の行をクリックするとやっぱり金額を入力する手間が省けるのよ。でも、質問文も含めて行全体がボタンになるのはあんまり美しくないかもしれないのよ。


適用方法は、
既存の SLG\SLG_SCOUT.ERB に、そのまま上書き
です。

https://github.com/wamekukyouzin/eratohoK/commit/2c18692955bda9517c148b2a4f44041bde84cd6e
の修正内容も含んでいるはずです。
