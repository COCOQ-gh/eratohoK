えーっと、人柱版が光栄にも取り込まれるとは思ってなかったので想定外のうｐです
「周回時あなた能力のリセット」は実装されたので緩和しても取り返しはつくとおもっています
製作者から訓練周り作り直す話題あったので、繋ぎとおもって気軽に上げますね
手元でいろいろ変更掛けてた版なのです。余計なトコも変わってます（下記）


（このパッチは）
K0.135基本仕様は　人柱版２の訓練バランスパッチ適用になってます
このままでは訓練での成長がシブいのでこれを緩和するものです

（変更点）
・能力６０以上での成長抑制が効きすぎて８０超えたあたりからほぼ成長なしに陥る状態を緩和
・逆に低レベル帯の成長を軽く抑制。　それでもK0.130基本仕様よりは成長しやすいの。
・「あなた」限定の訓練経験値の抑制は多少緩和。　
　ただし「あなた」がTALENT「超成長」を持つ場合、あなた限定の抑制は消滅します
・知略・政治・歌唱・料理訓練について、気力・精神力成長値を増加　
・勝手に好感度にかかわるソース量を変更。自分試用だったの。デバッグめんどいんでこのまま
　　あなたより能力が高いと「優越ソース」　低い場合は「屈従ソース」を加えます
　　好感度ガンガンあがる原因となる「歓楽」は抑えてます　斬りかかられ事案はふえたかも
・初回の訓練にバグがあったのこっそり修正（変数訓練者能力への代入が不適切）
