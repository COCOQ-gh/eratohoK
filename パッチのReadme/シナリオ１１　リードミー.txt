シナリオ１１　東方西欧録

ヨーロッパmapのデモシナリオです
歴史ゲーをたしなむkスキー向けマップです
map改造やシナリオ作成は大歓迎です

なるべくコンパクトに纏めようとしましたが、結果都市名が解りづらくなりましたね・・・
要望があれば、より分かりやすいマップに作り直したいです
ついでに新大陸も・・・



中継地点として、七つの海エリアがありますが、海に出られる都市は限られています。
逆に上陸はほぼすべての沿岸都市に行えます。※参照
時代的に海からの強襲上陸の有利を再現しようとしましたが、そのうち見直したいです

バイキング天子は、初期兵力が多い代わりに土地が痩せているので、サッサと襲撃しないと詰みます


そのうち史実に即したイベントなんかも盛り込みたいなぁ・・・


※注意
海エリア間の移動は不可、中継都市を落とさないといけない
例えばカファからアテネへ海路で向かうには、コンスタンティノープルを陥落させる必要がある
