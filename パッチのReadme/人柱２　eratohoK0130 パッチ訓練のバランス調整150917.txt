---[人柱２　150917追記]--
150916版からの変更点　（アウトプットの数値例は別ファイルに添付）
　・日常パートの訓練すべての基本経験値を　４から３．５に変更　　（COMFファイルのLOCAL:10で変更できるようにしてます）
    　→　低能力の育成スピードがちょっと減少
　・高能力の成長抑制を緩和
　　　→　ぜんぜん成長してなかったため
　・歌唱・料理の成長をさらに抑制。武闘より能力値２０早く抑制が加わる
　　　→　ABLUP関数にそういった趣旨がみられるため

------[追記ここまで]-----

　気になるあの娘を強くしたい！　
　そう思っていたはずなのに・・・気がつけば"あなた"が天下無双
　「違う、そうじゃない」
　そう思いつつ作ったものです

（概要）
日常パートの訓練コマンドでの獲得経験値について変更を加えたパッチです
本なしでも＋７点とか入ったりします。本あるとさらに倍。乱数に嫌われると経験点０もあり。
訓練相手がへっぽこだったり、自分の能力がすでに高いとどんどん目減りします
（内容）
・訓練結果に乱数の影響が入ります一定しません　（変更前は　失敗１、成功１、大成功２固定）
・獲得経験値に　訓練相手による効率を加えます
	格上（能力差４）以上の相手と訓練で効果増大　／　格下（能力差４）だと効果減少
・訓練者が６０以上の場合、成長しにくく
・"あなた"はさらにペナルティ　（訓練実行数がズバ抜けてしまうため）
