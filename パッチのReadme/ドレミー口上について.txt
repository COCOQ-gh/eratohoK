★このファイルはeratohoKのドレミーの口上試作です。(動作確認ver.0.135)

初心者のうえプログラミングの類が全く分からないのに挑んでいるので色々と穴だらけです。
またこのドレ―ミー口上はマミゾウ口上とJ咲夜口上を改変して作っていますので似たりよったりな内容になってます(特にマミゾウ口上に似ちゃってます)。
あと作り始めたのが0.130(現0.135)なので未だ予期せぬズレとかあるかもしれないです。
しかも推敲もまったく出来てない未完成状態です。
ついでに言うとライセンスがちゃんと出来ているのか不安です。


次は埋まってない思いつきで日常口上とエロネタを埋めていく
(戦闘中の口上にてプレイヤーとドレミーが一緒に居る時だけ出てくる口上って具合のコマンド？が出来たら嬉しいなぁって)


★導入方法
eratohoKのERBフォルダの中にある口上フォルダにこの「134 ドレミー口上」フォルダをぶち込む。以上。

★注意
この口上は「あなた(男)」を前提として作っていますので、それ以外のキャラだとシチュエーションに齟齬が出るかもしれません。また、この口上を導入することで調教難易度が変わるかもしれません。


★このドレミーについて

1.お姉さんだけど時に乙女チック(?)
2.夢の支配者だけど戦いの影響で夢の世界を支配出来ないでいる
3.獏は寝るけど夢は見ない
4.夢(願いの方)はお嫁さん(乙女ポイント)
5.政治キャラらしい(設定の方向音痴)

見切り発車なので細かい設定はないですし、勢い任せで書いてしまったので口上の中に矛盾が生じている可能性もあります。
