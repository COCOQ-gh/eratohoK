〓eratohoK ver0.130用 敵勢力補正＆教育環境是正パッチ

〓〓概要
いつもどおりバランス変更処理が入っていますのでご利用の際はご注意ください
・設定で「敵戦力への戦闘力補正」を追加
・詰め込み教育の廃止
・CFLAGの変更

〓〓パッチ内容
・CONFIG_SLG.ERB
　@CONFIG_SLG
　敵戦力への戦闘力補正(CONFIG:314)の追加、初期値は1(NORMAL)です
　原作の難易度名称を引用しています

・SLG_TURNEND.ERB
　@BATTLE_CALC
　敵戦力への戦闘力補正処理
　自勢力の部隊が戦闘に参加している場合、相手勢力に補正をかけます
　だいたい原作の難易度に近い補正を（想像で）かけているのでご注意ください

・SHOP_LIFE.ERB
　@CHILD_REARISING
　子育てのコマンド実行時に対象の子に行動済みフラグを追加

・CFLAG.CSV
　71,手引き誘導者
　72,教育方針
　に変更
　チェックした限り、教育方針以外の場所ではCFLAG:71の表記を、手引き以外の場所では教育方針の記述がなかったので、
　多分これで問題なく行くと思います（子供がいる状態でセーブするとちょっと齟齬が出るかもしれませんがご了承を）


〓〓おまけ
難易度調整については基本的にいろいろ用意して各人が好きな様に設定したらいいんじゃね的な考えでやってます
なので、よっぽどでなければ中の数字は変えません（変えるぐらいなら補正値の選択肢を増やす）

自分以外の子は会いに行った時点で教育モードになるので父親として会いに行く件はスルー
他の件についても作者の中の人がチェック始めているようなのでこちらもスルー
（フィルタ検索で拾いづらいのでテンプレ使って報告は書いてくれるとちょっと嬉しいです）

これでAI関係よみに行けるはず（フラグ）。
