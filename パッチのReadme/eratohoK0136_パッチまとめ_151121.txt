▼「一緒に泳ぐ」の成功判定と経験値について
　判定値が武闘＋(成長タイプによる補正 / 4)なのは変わらず
　・30未満：判定値／0
　・50未満： (判定値 - 30) * 3 / 2 + 50／判定値 - 30
　・70未満：判定値 + 30／判定値 - 30
　・100未満：100／判定値 * 60 / 100
　の確率で成功／大成功する（成功判定と大成功判定は別）

　キャラ二人がそれぞれ判定し、片方でも失敗すれば失敗判定、どちらも大成功判定に成功すれば大成功となる
　また、経験値は武闘が30未満であれば2, そうでなければ1を基本として失敗／成功／大成功で-1/0/+1となる
