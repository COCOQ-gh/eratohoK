▲概要
そういやRETURN 1をコメントアウトしたままだと不具合起こるよね（てへっ）

■パッチ内容
・SLG_AI.ERB
　@AI_DIPLOMACY_ACT
　条約締結時にフラグを立てておき、そのフラグを返り値に使うように

・SLG_AI_INVATION.ERB
　@SLG_AI_CREATE_INVASION_TARGET_LIST
　すでに部隊を送った都市に対する追加は兵の確率を少し上昇（都市侵攻による経済ダメージを考慮して波状攻撃を選びやすく）
　この辺はそのうちAI毎に選択率を変えるようにしようとは思う（RANDに与える数を配列から読み込ませるだけの簡単なお仕事）


▲パッチ適用方法
eratohoK 0.161に「周回できない不具合修正パッチ」を適用の上、
そのまま上書きしてご利用ください
