・MAP.ERB
　ようは地図中の地域グループ名（妖怪の山とか）を斜体[ FONTSTYLE 2 ]にして区別しやすくしてみました
　直後で斜体を解除[ FONTSTYLE 0 ]すると激しく欠けるので、
　わざと一文字分だけ後ろに欠けても問題なさそうな罫線やスペースを含ませて斜体にしてあります
　
　例：
　FONTSTYLE 2
　PRINT 妖怪の山─
　FONTSTYLE 0

　FONTSTYLE 2
　PRINT 魔法の森┐
　FONTSTYLE 0

（[PRINT 天界┘]　辺りとかちょっと変か？）　

　一応色々な等幅フォントでチェックしましたが多分ずれないとおもいます（MSゴシックなら大丈夫）
　太字+斜体[ FONTSTYLE 1 + 2 ]はフォントによっちゃズレるのでおすすめしません
