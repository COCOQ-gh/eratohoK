妖怪vs人間vs山の三つ巴に月＆地獄が横槍を加えるイメージです

マップ南側は弱小勢力が入り乱れた戦国時代
マップ北側は比較的スカスカです


本作の見所さんとして、戦闘が可能になる7ターンまでにランダムイベントで毎回勢力図が微妙に変化します
大連合同士のシナリオだと展開が同じになりそうだから取り入れました
が、そっちがメインになってしまったので大連合要素はそんなでもないです
見所さん？！まずいですよ！
