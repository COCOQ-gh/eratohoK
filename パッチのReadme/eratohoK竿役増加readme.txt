# eratohoK ver0.110用 竿増加パッチ

## このパッチについて
せっかく霖之助や妖忌や雲山が居るのに、調教ではアナルを掘れるだけで幻想少女を一緒に犯すことが出来なかったのをコマンド追加で対応しました。これで誰が父親か分からない子供を和姦で量産出来るぞ！

* 「代わる代わる挿入」「代わる代わるＡ挿入」コマンドを追加。
	* 順番に挿入と違って複数で一人を犯すことも出来ます（二穴責めとか三穴責めが来てくれれば不要に…）
	* 分岐量が凄い事になるのでペニスもペニバンも竿に統一してテキスト処理を省力化（追記してくれる人募集中です）
	* 挿入対象の相手役はランダムで竿役一人だけに設定しているのでコマンドごとにSOURCEにバラつきが出ます
* 「親友を付ける」コマンドの条件文が正しく動作していなかったのをついでに修正
