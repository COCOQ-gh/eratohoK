
eratohoK 0.232.1用 @SELECT_CHARA_LIST_MULTIに、キャラクターの情報確認機能を追加するよパッチ

独立時に誘うキャラクター
および
周回時のランダムキャラや子供の引き継ぎ
の選択画面では7種類の能力値が表示されてるけど、もっと色んな情報を確認したい！という人向け。
（あらかじめ、キャラクターの情報を確認しておいてから独立なり天下統一なりすればいいんだけどね。）

料理の数値の右隣に ** が表示されているので、それをクリックすれば普通にキャラクターの情報を見ることができます。
クリックではなく、キャラクター左端の[]で囲まれた数字に10000を加算して入力しても同じです。


適用方法は、
既存の SYSTEM\SELECT_CHARA_LIST.ERB に、そのまま上書き
です。


不具合や副作用があるかもしれないけど、私（パッチ作成者）は修正しません。
とりあえず、選択ロジックが "" じゃない場合には変になるかもしれない。



※このバッチは本体に取り込まれた場合はパッチ側の権利放棄となります
ということにしておくけど、取り込まれたほうがいいんだろうかどうなんだろうか。
本当はCALL SHOW_INFO_WITH_UIする直前でSHOP_CHARA_LISTを設定するんじゃなくて、@SELECT_CHARA_LIST_MULTIの最初のほうで設定しておいてキャラクター一覧表示でも利用するほうがいいのかなあ。
あと、ボタンが ** というのもどうなのかなあ。
ERBファイルの記述も含めて、全体的に美しくないような気がするなあ。
eraで利用するなら、使う人が自由な感じでアレコレすればいいと思うよ。

まあ、非公式機能のお試しデモ版のつもりで！
