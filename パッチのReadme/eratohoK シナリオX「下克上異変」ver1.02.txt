eratohoK1追加シナリオ　「下克上異変」

このパッチは、eratohoK用のシナリオパッチです
・ファイル名のXXX
・及び各関数名のXXXを該当するシナリオ番号に変更
・更に、eratohoKのNEWGAME.ERBファイル内、@SELECT_SCENARIOに、このシナリオの選択肢を追加
以上を各自で設定する事で動作します(バリアントに組み込まれた場合はその必要はありまん)
※SLG_ESCAPE.ERBを上書きしますので、マージされていない他パッチとの競合にご注意ください
　特に「K0.122バニラ用MAP拡張パッチ」には今の所“未対応”ですので、そのまま上書きはしないでください“確実に「地名違いエラー」が発生します”

シナリオの内容は、幻想郷で下克上が相次ぎ、各勢力のトップが部下達の手で監禁、調教されてしまうというものです
既存のシナリオと違い、強キャラ達は軒並み捕虜となっています
また、「あなた」が全国統一を果たした場合、異変の原因である「あの子」が挙兵します
このシナリオに外来人は登場しません

2015/08/19
・敵国を滅亡させた時、その国で下克上されていた元当主を便乗senkaするか救出するか選べるように
・下克上された元当主がいつまでたっても堕ちなかったのを修正
・それに伴い、国外捕虜が陥落するまでに必要な従属度を-500から-1000に増やし、一回の調教で減少する従属度を緩和。
・上海人形軍の初期設定捕虜に神綺さまと魅魔さまを追加、ついでに雛とメディスンとる〜ことをメンバーに(人形つながり)
・各勢力の友好度も若干変更
・下克上による特定カップリング調教口上を追加
　・さなかな　　君主早苗　捕虜神奈子
　・ちぇんゆか　君主橙　　捕虜紫

・自国捕虜が他国の君主に勝手に調教されるバグを修正
・敵国滅亡時、その国で下克上されていた元君主が、謎の逃避行するバグを修正
・下克上による特定カップリング調教口上を追加
　・うどんげっしょー　　君主うどんげ　捕虜えーりん
　・こいここ(確率低)　　君主こいし　　捕虜こころ(こころがこいしの捕虜にならない限り発生しないので極めて低確率)

・プレイが可能な状態に到達
・捕虜と調教者の自動選択が総当りでやっているため、環境によっては重くなるかも？


今後の予定
・当主にも調教対象にも竿無いはずなのにどっちかの子供が増える謎のバグ。原因不明。
・拡張マップ及び旧作勢力の実装に合わせ、このシナリオも更新予定(旧作キャラの上下関係とか二次創作モリモリになりそう)
・「あの子」反乱後、特定の拠点を解放すると捕まっていた仲間を救出できるイベント作りたい
　たとえば守矢神社奪還すると神奈子さま救出できたり、紅魔館解放するとレミリアが救出できるみたいな
・外交調教で当主を骨抜きにすると、それまで陵辱される側であった元当主が牢を破ってクーデターを起こし、下克上返しするイベントとか
　例えばこのシナリオだとサニーが当主やってて霊夢ちゃんが陵辱されてるが、サニーが外交調教でアヘると霊夢の逆襲が始まるみたいな
・捕虜と調教者の特殊な組み合わせによる専用口上がめっちゃたくさん残ってるので、チョロチョロやっていく
　というか口上作者さん書いてくれてもいいのよ？いいのよ？！
・ぶっちゃけロマンで作ってるのでバランスとか度外視です、それでも要望あれば修正していきます
・不具合は見つけ次第やっつけちゃおう
