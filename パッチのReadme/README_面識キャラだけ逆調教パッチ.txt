◇eratohoK Ver0.166用 面識キャラだけ逆調教パッチ

■パッチ概要
　逆調教時に面識のあるキャラしか参加しなくなるコンフィグを追加しました。
　[交流パートに関する設定]から利用してください。
　
　CONFIG:17を利用しています。

■パッチ内容
・CONFIG.ERB
　@CONFIG_OTHERS
　　コンフィグ項目を追加

・SHOP_LIFE_REV.ERB
　@SELECT_REV_TRAINER
　　参加者選択時の分岐を変更



■導入方法
　eratohoK Ver0.166用 外交調教強化パッチ 修正版(eraT0008666)まで順に適用した上から導入してください。
