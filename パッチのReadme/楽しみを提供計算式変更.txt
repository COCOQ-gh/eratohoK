楽しみを提供したときの効果が、キスがやたら強いなーと思ってERB見たら
1.満足度の計算にABLよりも強くなりやすいEXPを使用している（提供元）
2.満足度の計算にABLよりも強くなりやすいEXPを使用している（提供先）
3.相互の性別を問わない（当然だけど）

こりゃ強い訳だ。数値だけが目的ならキス以外を選ぶ理由がない（例え提供元だけでも十分過ぎる程）


という訳で大雑把ですが改変しました。全体としてはまぁまぁ強く、キスはかなり弱く
平均化しつつも、絡む要素を弄ってコマンドに個性を持たせたい、と思いながら弄ったけど上手くいってるかなぁ
