このパッチは、eratohoKの敵勢力ＡＩに、シナリオ側からの命令でデフォルトにない特殊な行動を取らせる事を目的としたパッチです
ＡＩの行動は即ちゲームバランスに直結するため、特殊行動の命令は極めて限定的な条件を前提としてください
　例　正邪軍の残り拠点が一つしかない場合、イベントの後、全兵力を結集した決戦部隊を拠点上に出現させ、解散させずに維持し続ける。
同封のシナリオ6「下克上異変」での正邪軍最終決戦イベントを正しく動作させるには、このＡＩパッチを使用してください(なくても遊べます)

使用FLAG
FLAG:101　　　このフラグが1以上になると、“拠点が一つしかない全ての勢力に置いて”部隊解散をしなくなる
　　　　　　　今後の拡張の可能性を考慮し、使う場合は「特殊行動を起させたい勢力の番号」を入力するのがベスト

このパッチの改造について
　自由に改造、投稿して構いません。
　ただしＡＩの行動が変るため、ゲームバランス等について、必ずバリアント制作サイドに相談をした上でアップしてください。
