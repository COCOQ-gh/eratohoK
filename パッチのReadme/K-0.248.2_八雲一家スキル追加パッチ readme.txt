八雲一家のスキル追加パッチ。橙が2面ボス扱いのみだったから、Ex中ボスとしての能力も欲しくて作った。藍様・紫様はそのまま流れで（固有スキル数、橙の方が多く成るのに抵抗もあった）。

難易度保持のため追加スキルは全て初期不可、ショップ可、習得デイリーなし。習得デイリー無の理由は手抜きとも言う。

おおよそのスキル説明

橙
青鬼赤鬼：武闘差依存の確立で敵全体を暗殺。発動率は高いが、発動後別個に成功判定(各5〜80%)があるので発動してもなにも起きないことも多い。敵との武闘差が広がりすぎるとぶっ壊れな性能に成るかも。
飛翔毘沙門天：武闘依存の連続攻撃。乱数に大きく依存する。初期ステでは300〜2700ダメ。武闘100で750〜6000ダメ。ダメージ幅がやや広すぎるかもしれないが、連続攻撃なので回数が増えるほど最終ダメージ幅は落ち着くはず。

藍
八千万枚護摩:50%と被害軽減系では郡を抜いた発生確率を誇る。反面、効果量は僅かに3%と知れている。安定感はあるがいまいちパッとしない。（期待値だけならこいしと並んで同率１位）

紫
肉体分解機：橙の飛翔韋駄天の完全上位互換。同等の発生確率とより高い火力を誇る。その代わり伝達不可。隙間を操れるのは紫様だけなのだ。
