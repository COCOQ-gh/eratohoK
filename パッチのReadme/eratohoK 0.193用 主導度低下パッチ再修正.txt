eratohoK 主導度低下パッチ再修正


ボーナスが付く素質を項目ごとに切り分け、長期間の調教では数値がほとんど変動しなくなるように。
複数勢力をたらい回しにでもされない限りは以前のように主導度-1000が並ぶことはなくなりました。


変更点

+主導度Ｎ、主導度Ｕ、倒錯度でそれぞれ倍率が変わる素質を変更。キャラごとの差異を大きく。
+捕まりっぱなしだと数値はほとんど変動しないように。既に他勢力に陥落している場合も変化を小さく。
+特殊勢力の調教による変動は主にFUCK_SP内で行い、FUCK_EXP_PROCESSでの変化量は特殊勢力に捕縛中、もしくは所属していると1/5に。

主導度Ｕ
　低下量加算…Ｍ気質、臆病、痛みに弱い、貞操観念、悲観的
　低下量半減…気丈、貞操無頓着、プライド低い、感情乏しい

主導度Ｎ
　低下量加算…Ｍ気質、倒錯的、解放、快感に素直、中毒しやすい、臆病、痛みに弱い
　低下量半減…気丈、自制心、感情乏しい

倒錯度
　低下量加算…Ｍ気質、倒錯的、解放、快感に素直、中毒しやすい
　低下量半減…自制心、感情乏しい、無関心
