②CONFの可能条件追加

◆二穴可能体位追加--------------------------------------------------------------------------------

	Ａ背面騎乗位中の相手に　正常位が可能に　⇔　正常位→Ａ背面騎乗位が可能に
	Ａ背面騎乗位中の相手に　対面座位が可能に
	Ａ背面座位中の相手に　背面騎乗位が可能に

　具体例：Ａ背面騎乗位でマンぐり返しっぽく双成魔理沙に犯されてる霊夢のＶに、あなたが正常位で挿入可能に
　　　　　(薄い本で良く見かける形のサンドイッチなのに、今まで出来なかったという)

◆フェラ-------------------------------------------------------------------------------------------

　以前は「挿入中は1-1のみ可能」(挿入れられてる双成や男キャラのペニスを咥えられるのは一人だけだった)
　　→条件文に性技が２以下の場合、を追加（フェラ参加者が全員性技３以上ならＷフェラ、Ｔフェラが可能に）

　具体例：あなたが双成レミリアを後ろから犯してる状態で、咲夜だけじゃなく美鈴やパチェにも
　　　　　一緒にレミリアのペニスを咥えさせられるように（要性技Ｌｖ３以上）

◆足コキ-------------------------------------------------------------------------------------------

　ターゲット(足コキされる側)が何かしらの性交中は全て不可だったのを
　(Ａ版を含む)後背位・背面座位・騎乗位・背面騎乗位されている時なら可能に変更
　(それ以外の体位やターゲットのペニスが使用中の時は不可)

　具体例：フェラの具体例の足コキ版。あなたにゴリゴリされながら三妖精にふみふみされる双成阿求とか？

◆足を舐める---------------------------------------------------------------------------------------

　ターゲット(足舐めされる側)が何かしらの性交中は全て不可だったのを
　(Ａ)背面座位されている時なら可能に変更

　プレイヤーからターゲット／ターゲットからプレイヤーに足舐め中なら不可　を
　ターゲットからプレイヤーに足舐め中なら不可　のみに変更。

　具体例：足をしゃぶられてくすぐったさで涙目の双成リグルんを足コキするゆうかりんとか、ええやん？

◆土下座-------------------------------------------------------------------------------------------

　　条件文を修正し土下座ックスがちゃんと出来るように。
　（土下座する方・される方がどちらかでも性交中だと不可能になっていた）

　具体例：あなたが土下座してる白蓮を後背位で犯す　⇔　（犯している状態で土下座させる）

　　　　　あなたと紫が背面座位でイチャつきながら、
　　　　　依姫が陥落した双成豊姫にＡ後背位されてる状態で紫に向けて土下座　など
