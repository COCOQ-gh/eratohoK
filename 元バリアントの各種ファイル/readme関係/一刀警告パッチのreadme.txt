＜内容＞

1.一刀によるＮＴＲへの警告文表示を追加
　
　一刀に手を出されているキャラの惚れ度が一定以上高くなった時に警告文が表示されるようになります

　警告は２段階になっており、それぞれ一度だけ表示するようになっています



2.一刀への恋慕・恋人取得時と、警告文の色変更

　一刀への恋慕・恋人取得時と、警告文の色を変えて確認しやすくします



3.CFLAGの追加

　未使用の『CFLAG:90』を警告文表示が毎回表示されないためのフラグ使用しました

　『era恋姫 変数表』に『90:一刀への惚れ度の警告フラグ』を書き加えています
