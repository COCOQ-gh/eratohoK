;-------------------------------------------------

　CFLAG:300とCFLAG:301を使用し、ランダムキャラの一種である誰かの子供の口上を作成しました。
　現状だと一刀の子供だろうとふたなり華凛様の子供だろうと、あなたの子供として判定されると
か、色々問題はあるのですが。

　取りあえず最初に性格を設定できるようにしたので、CFLAG:301の値を増やすことで複数の性格の
娘口上が作成できる、はずです。たぶん。

　現状では元気な”ボクっ娘”なる性格を実装しております。口上は虫食いだらけですし、一部口
上で通常ランダムキャラの台詞が表示されますが。そのあたりはおいおい直していきたいと思います。

　バグや妙な挙動があればお教えいただければ幸いです。

　read meのテンプレートはランダム口上を作成された方のものを借りさせていただきました。
　この場でお礼申し上げます。
